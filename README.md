Container orchestration and cloud-native applications.

Part of [Train Station](https://github.com/jon-the-dev/saas-train-station), a tiered
learning platform. This repo holds the **free** lessons for this track. **6 additional
premium lessons** — EKS Workshop, Istio Service Mesh, Kubernetes the Hard Way,
Kubernetes Patterns, and more — are available in the Train Station app.

## Lessons

| # | Lesson | Type | Level | Source |
|---|--------|------|-------|--------|
| 1 | [Introduction to Kubernetes (edX – LinuxFoundation)](lesson-01-introduction-to-kubernetes-edx-linuxfoundation/) | Lab | Beginner | [Linux Foundation](https://www.edx.org/course/introduction-to-kubernetes) |
| 2 | [Kubernetes Official Tutorials](lesson-02-kubernetes-official-tutorials/) | Lab | Beginner | [CNCF](https://kubernetes.io/docs/tutorials/) |
| 3 | [Helm Charts – Artifact Hub](lesson-03-helm-charts-artifact-hub/) | Repo | Intermediate | [CNCF](https://artifacthub.io/) |
| 4 | [Kustomize Examples](lesson-04-kustomize-examples/) | Repo | Intermediate | [CNCF](https://github.com/kubernetes-sigs/kustomize) |

Each lesson is a self-contained guide (overview, prerequisites, key takeaways, and a
step-by-step walkthrough). Open a lesson directory to read its `README.md`.
