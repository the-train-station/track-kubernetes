# Kubernetes — Train Station Track

[![free-only](https://github.com/the-train-station/track-kubernetes/actions/workflows/free-only.yml/badge.svg)](https://github.com/the-train-station/track-kubernetes/actions/workflows/free-only.yml) ![Free lessons](https://img.shields.io/badge/free%20lessons-6-blue) [![License: CC BY-SA 4.0](https://img.shields.io/badge/License-CC%20BY--SA%204.0-lightgrey.svg)](LICENSE) ![Last Commit](https://img.shields.io/github/last-commit/the-train-station/track-kubernetes) ![Issues](https://img.shields.io/github/issues/the-train-station/track-kubernetes)

Container orchestration and cloud-native applications.

Part of [Train Station](https://github.com/jon-the-dev/saas-train-station), a tiered
learning platform. This repo holds the **free** lessons for this track. **6 additional premium lessons** — Kubernetes networking, security and RBAC, EKS production operations, Kubernetes the Hard Way, GitOps, and observability — are available in the premium track.
